    #游戏/简评

    《波西亚时光》正式版评测9分 白手起家的工坊主传奇 ~ 停不下来
    游民星空 GamerSky.com

    #游戏/推荐人群

    模拟经营游戏爱好者，种田游戏爱好者

    #游戏/优点

    • 魔性的模拟经营
    • 充实的RPG要素
    • 丰富的游戏玩法
    • 有意思的世界观
    • 附送一份《模拟人生》

    #游戏/缺点

    • 配音和音效比较糟糕

    　　恰好在一年以前，我评测了《波西亚时光》的抢先体验版。彼时的它虽然还不完整，但出色的玩法已经显露出了极大的潜力。经过一年的紧密开发，《波西亚时光》的正式版终于在1月15日面世了，而我也有幸提前做了体验。可以说，相比测试版，现在的它变得更加丰富、更加合理、也更加的好玩。如果你也是一个模拟经营类游戏的爱好者，那你绝对不应该错过这款精彩的作品。

    视频评测：

    种田游戏也能后启示录？

    　　说出来你可能不信，严格来说，《波西亚时光》其实是个科幻作品。

    　　很奇怪对吧？一个和《星露谷物语》、《牧场物语》类似的种田游戏，怎么就成科幻作品了呢？

    　　其实，开发商给《波西亚时光》安排了一套特别完整的世界观，而这个世界观居然设定在了末日之后。游戏里的人类曾经创造过非常高科技的文明，可是因为一次全球性的战争，地球被弄得满目疮痍，一时间黑云蔽日，民不聊生。人类文明最终被自己一手创造的科技给毁灭。

    view

    　　经过了数百年的黑暗时代之后，一个叫做皮奇的大英雄找到了驱散黑云的办法，带领仅存的人类重新走向了光明。经过重建之后的地球，版图发生了非常大的变化，而故事的舞台波西亚则偏居一隅，远离世界的中心。这里就像是一个世外桃源，小镇的居民安居乐业、悠然自得。而玩家则需要扮演一名外地人，乘船来到这里，继承父亲留下的破旧工坊，游戏也自此拉开了序幕。

    view

    　　这样的世界观，绝不仅仅只是编个故事给你听而已，它实际上渗透进了游戏的方方面面。比如你在挖矿的时候，很可能就会挖到许多奇怪的东西：数据光盘、芯片、或者微型电路板之类——这些玩意儿实际上都是战争时代的遗物，你要组装的许多设备，都离不开它们。可是因为时代久远，制作它们的方法大多都已经失传了，所以你只能寄希望于从土里把它们发掘出来。

    view

    　　又比如，你能在小镇里遇到两伙暗中较劲的势力，其中之一是主张兼收并蓄，从前人科技中取其精华去其糟粕的科研中心，你可以把挖到的数据光盘给他们，让他们破译出新的组装图纸；另外一个则是完全反对前人科技，主张让大家反朴归真的光明教会，你可以把挖到的那些史前“危险品”交给他们销毁，换取一些珍贵的植物种子，或是其他用钱买不来的稀奇玩意儿。

    view

    　　人们的信仰、小镇的地标、敌人的设计、故事的发展……游戏的每一个细节，可以说都建立在这样一种后启示录风格的世界观之上。所以别看这个游戏的画面这么小清新，内容这么休闲，说它是个科幻游戏，可一点儿也没错。

    　　而这样的世界观，实际上也给了游戏一个强有力的暗线。黑暗时代究竟发生了什么？科学派和教会派之间到底会不会爆发冲突？波西亚小镇的和平会长久地持续下去吗？人类会再次陷入到战争当中吗？《波西亚时光》绝不是一个大而空的沙盒，它被填充进了非常多的线索与悬念，驱使着你去揭秘、去发现。你不会觉得每一天从游戏里醒来都无所事事，或者非常被动地去完成那些工坊的订单；相反，你会无比期待新一天的开始，你的内心会对这片神秘而广袤的大陆感到非常好奇。

    view
    波西亚世界的版图

    　　再退一步说，这样的世界观其实也非常合理。你想啊，要不是因为战争使得人类的文明倒退回了几百年前，否则谁整天没事干，跟着你面朝黄土背朝天啊？从这个角度来说，世界末日反倒是个非常必要的设定呢。

    不是牧场物语，而是工坊物语

    　　既然是模拟经营，《波西亚时光》难免会拿来和《牧场物语》或《星露谷物语》来比较。但实际上，相比后两者更偏向农业的游戏内容，《波西亚时光》则更偏向工业一些——毕竟，你从父亲手里继承来的不是某某农场或者某某牧场，而是一家工坊。从最初的手工产品，到后来的大炼钢铁，看着自己从家徒四壁的穷小子变成名震一方的富商大贾，这种白手起家的快感，相信也是许多模拟经营爱好者都为之痴迷的重要因素。当然啦，农业也是《波西亚时光》非常重要的一环，只不过一开始的时候比重并不大罢了。随着游戏的进行，种植业、养殖业与畜牧业都会逐渐向你敞开怀抱。到了后期，你甚至还能驯化一些野生动物，让它们成为你的坐骑。

    view

    　　游戏的主线内容其实非常简单，接订单，生产所需商品，赚取酬劳、NPC好感与工坊评级，然后继续扩大生产，殷实自己的家底。除了短时效（5天到10天不等）的订单之外，还有周期性的大订单等待着你去完成，比如建造一座木桥，或者建造一辆三轮车——这些订单一般都没有时效限制，你可以慢慢建设自己的工坊，等到心有余力的时候再来处理这些事情。所以你会发现游戏的体验非常充实，小订单与大订单相互交错，基本上每一天你都能有事儿干。

    view

    　　那么，该如何获取订单所需的原材料呢？就像《我的世界》那样，随处可见的树木、石块就是最好的资源——而且，这些东西都是会重复刷新的，要是你把家附近的树都砍光了，过两天它们还会再长出来的。

    view

    　　除了树木和石块，游戏里还有一些遗迹，在交了通行费之后，你可以在里面挖到诸如铜矿石、铁矿石这样的矿产，也能借助探测器挖到电路板这样珍贵的史前遗物，甚至还有文物的碎片——集齐碎片之后，你可以去科研中心把修复它们，然后摆在自己的家里，不仅美观还能有能力上的加成。

    　　除了普通遗迹之外，还有一种叫做危险遗迹的地方。它其实相当于是一个副本，你要在里面闯过好几关，并且和许多怪物战斗，最后还会经历一场BOSS战——是的，战斗在这个游戏里同样重要，而且是必须要经历的步骤，因为许多稀有的原材料只会在危险遗迹里出现。

    view

    　　其实一般来说，模拟经营游戏的战斗基本只是给你提供一个获取原材料的途径而已，因此大家都不会花太多的精力去设计诸如战斗系统、或者敌人的攻击方式之类的东西。《波西亚时光》也有着类似的考虑，比如那些野生动物（熊猫蝙蝠、彩虹草泥马什么的，都特别好玩），它们基本上都毫无挑战性，虽然会攻击你，但无脑平A过去，你就能收集到它们身上你想要的材料。

    view

    　　然而危险遗迹却不同，这里面的怪物相当有难度，不同的敌人有着不同的攻击方式，可能需要灵活地运用翻滚等技巧才能通关。虽然动作系统与打击感等方面还很稚嫩，但作为一款模拟经营游戏来说，这或许是可以被接受的，就看你自己的耐受度如何了。

    　　我想，这样的设计或许会劝退一部分只想轻松经营工坊，不想费劲打怪的休闲玩家——但我们也需要注意到，《波西亚时光》其实引入了非常多的RPG要素，通过技能树、等级、以及装备的提升，很容易就能利用数值碾压打过那些副本。而且在通关过一次之后，你就可以付钱给民兵团，让他们代你去探索危险遗迹了，而你只需要在家安心负责生产就好。看得出来，开发商仍然是想在提高技巧要求的同时，尽量降低整体难度的——毕竟，它本质上仍然是个模拟经营游戏嘛。

    view

    　　值得注意的是，《波西亚时光》里是有着体力值的设定的。每挥一次斧头，每敲下一次十字镐，都会耗费相应的体力。如果归零，除非坐下来或者吃食物来恢复，否则就只能睡一觉，等待第二天早上自动补满了。

    　　体力值的设定，一开始可能会让你感到拘束。但相信我，只是一开始的时候会这样而已。随着日后角色的升级和装备的提升，你的体力值会迅速地成长起来。再加上后期原材料会有大量的盈余，而你每天要做的其他事务变多（例如社交和管理工坊，这些都是不耗费体力的），所以体力值绝对不会成为你的掣肘，相反可能还用不完。

    view
    好消息是，这个游戏没有饥饿值和物品损耗度的设定，全然不会让这些东西影响到你的游戏节奏，实在是太棒了

    　　和测试版相比，正式版的“肝度”虽然也不低（这种类型的游戏，很少会有不肝的，相信我……），但比之前要好了一些。至少原材料的获取难度降低了不少，下两次矿，就能积攒到数百个各种各样的矿石，够你冶炼上好几天的了。中后期你压根不需要天天待在地底下，安心把自己的工坊安排好就行。当然，如果官方能给生产制造环节营造出更多的正面反馈，我想体验会更加完美。比方说，适当地降低某些产品的生产时间，又比如说，让挖矿这个过程更加有趣，更加具有变数。

    view

    　　除此以外，正式版也加入了更多的“突发事件”。一觉醒来，你可能会突然发现有人来找茬、大家的东西神秘失窃、或者是小镇的水遭到了污染——这个时候你就需要和镇子里的人一起解决这些事端。可以说，除了种田游戏一般都会有的乐趣之外，这些或主线或支线的任务，同样也扮演着重要的角色，它使得游戏的节奏更加紧凑，也减轻了模拟经营游戏所带给你的“工作感”与“重复感”。

    view

    是模拟工坊，更是模拟人生

    　　在《波西亚时光》这款游戏里，你除了要经营好父亲留下来的旧工坊，你更得经营好自己的人生。

    　　除了通过完成订单等方式来提升自己之外，和镇子里的其他人搞好关系显然也是非常重要的一环。交谈、送礼、切磋武艺、猜拳嬉戏……等和NPC变成朋友之后，你还有可能会开启一条隐藏的任务线，让你对这些陌生人有更加深入了解的机会。

    view

    　　当你们的好感达到一定程度，你还能向他告白、约会、最后私定终身（男男或者女女也是可以的）。成为你的爱人之后，不同的NPC还会有不同的“福利”，比如护士姐姐会为你的针灸打折，而聪明能干的农家女艾米丽则会时不时来你的工坊帮忙。

    　　这还不算完，你甚至还能和你的伴侣诞下爱的结晶——你说同性伴侣怎么办？别担心，你可以向光明教会领养一个宝宝，性别从来不会阻止你成为一个好父亲或是好母亲。

    view

    　　和测试版相比，正式版在保留了原先那些各具特色的人物之外（比如非常搞怪的“葫芦七兄弟”），还加入了一些新的NPC，譬如一个从一开始就看你不顺眼的死对头，还有前段时间刚刚宣布的由著名主播女流配音的流浪歌手（可惜我玩的那个版本还暂未加入她），波西亚的生态比以往更加地丰富了。

    view

    　　所以，这实在是一个大有可玩的游戏。除了正常的生产经营之外，你还能装扮自己的房屋，你还能和镇子上的人一起欢度佳节，你还能体会恋爱约会的甜蜜，以及承担为人父母的责任。在《波西亚时光》里，你真的不必担心没有新东西可以体验，它总能给你带来惊喜。

    view

    一步之遥

    　　其实我对《波西亚时光》的绝大多数部分都是相当满意的，只是有些小的瑕疵，我不得不单独拎出来讲。

    　　首当其冲的就是配音问题。其实这个事儿早在测试版评测的时候我就已经提出来了，我发现游戏里NPC的配音质量参差不齐，有些还说得过去，而有些明显就是业余选手配出来的（比如女性主角的其中一种声音）。甚至于，你还能听出来录音设备和录音环境的差别，一些NPC的对话似乎是用非常劣质的麦克风，在非常不专业的环境下录制下来的。哪怕做了后期，也和别人的配音格格不入，听上去相当粗糙，很影响体验。

    view
    女主角的录音效果真的不算好

    　　除了配音的问题以外，游戏的音效也算不上好，特别是战斗时攻击敌人时的音效，给了我一种“劣质手游”的感觉。很遗憾，这些缺点并没能在正式版里得到修正。虽然这并不会影响游戏的核心玩法，但会显著拉低游戏的印象分。

    　　其次，《波西亚时光》里的组装与合成基本都有详细的说明书可查，这一点非常棒，总算不用经历《我的世界》那种“入门即抓瞎”的阶段了。但是，我发现哪怕在游戏很初期的时候，都有可能会出现一些需要有一定设备积累才能生产的订单需求。我不知道这是Bug还是有意为之，总之在接订单之前，你最好先查一查随身携带的组装说明书，以免出现生产不出来，不得不毁约的情况。

    view

    　　最后，虽然游戏也支持手柄，但根据我的体验，手柄的操作似乎并没有键鼠那么方便。翻页、组装、合成、调整背包，都是鼠标的操作更为直观。这当然算不上什么大问题，但可能会对主机版的体验产生一些影响。至少，希望游戏日后登陆NS平台的时候，能加上触控屏的一些操作，或许能够让手柄的操控更有效率一些。

    view

    　　至于联机模式，目前官方并没有明确一定会加入，只是说根据玩家的反馈再做考虑。模拟经营类游戏确实很适合一些轻度的联机要素，但至少对我个人来说，目前纯单机的《波西亚时光》，它丰富的内容已经足够让我玩上很长一段时间了。如果你对联机模式看得很重，那么我推荐你可以等一等；但如果你单机模式也能玩得很开心，那它就不会是什么问题。

    view

    结语

    　　我很享受在波西亚度过的时光：日出而作，日落而息，和各色各样的人相识，期待每一天都能发生一点不一样的事情。坦白说，在现实生活中，你很难去经历这样一种面朝大海春暖花开的恬淡生活，而《波西亚时光》则用一种特别美好的方式，让你在虚拟的世界里，体会了一次别样的人生。

    　　就像之前我给测试版写过的那篇评测那样，其实整篇文章我都没有提过《波西亚时光》是我们中国的开发者制作的作品。因为在我看来，哪怕褪去了国产游戏这层保护色，《波西亚时光》也依然是一款足够优秀的模拟经营类游戏。所以我完全没有必要提它的身份，它值得我毫无保留的欣赏与喜爱。

    view
    我就是工坊主传奇！
